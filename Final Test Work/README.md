# Данная программа позволяет пользователю: 

- Ввести количество элементов массива;
- Ввести максимальный размер выводимых строк;
- Ввести элементы массива;

После заполнения массива, производится вывод изначального массива на экран. 
Далее вычисляются строки, длина которых меньше или равна максимальной длине строки.
И, наконец, на экран выводится массив с подходящими строками.
